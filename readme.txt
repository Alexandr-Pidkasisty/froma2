Папка froma2 является корневой папкой репозитария библиотеки froma2.

Структура и описание файлов и папок репозитария
-----------------------------------------------

LICENSE
Файл с лицензией на библиотеку froma2

readme.txt
Текущий файл, который Вы сейчас читаете.

.gitignore
Служебный файл репозитария со списком исключений.

changelog.log
Файл с историей коммитов.

CMakeLists.txt
Конфигурационный файл системы построения проектов CMake.

cmdbuild.cmd
Пример кроссплатформенного командного файла, автоматизирующего построение проекта библиотеки с примерами.

winbuild.cmd
Пример командного файла для операционной системы Windows 10, автоматизирующего построение проекта библиотеки с примерами.

libsource
Папка с исходными кодами библиотеки FROMA2 и конфигурационным файлом системы CMake, обеспечивающим компилирование и сборку библиотеки; основная папка проекта.

doc
Папка с документацией библиотеки FROMA2.

examples
Папка с примерами, упрощающими знакомство с инструментами библиотеки FROMA2; каждый пример располагается в отдельной подпапке; в этой же подпапке располагается собственный конфигурационный файл системы CMake для этого примера.

build
Папка отсутствует в репозитарии, но создается на машине пользователя при запуске командного файла cmdbuild.cmd или winbuild.cmd из локальной копии репозитария; в подпапке build/bin создаются исполняемые файлы программ с примерами; в подпапке build/lib создается скомпилированный файл библиотеки FROMA2 с именем “libfroma2.a”. При сборке проекта без использования командного файла cmdbuild.cmd или winbuild.cmd рекомендуется придерживаться этой структуры папок.

Исходный код программы для демонстрации работы класса LongReal_test.
При компиляции и сборке программы исполняемый файл будет носить то же имя, что и данная папка. Рекомендуемое расположение исполняемого файла - папка build/bin.

Макрос #ifdef PRINTREPORT перенаправляет вывод с экрана в файл.
Полученный таким образом файл с результатами расчетов располагается в папке
froma2\build\examples\LongReal_test\output и называется Report.txt.

Таблица проверки любого оператора содержит следующие колонки:
- первая колонка - первый операнд в операции; число типа longReal;
- вторая колонка - знак операции;
- третья колонка - второй операнд в операции; число типа longReal;
- четвертая колонка - результат операции;
- пятая колонка - результат операции с теми же числами, заданными типом double.

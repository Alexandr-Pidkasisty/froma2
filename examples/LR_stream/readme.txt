Исходный код программы для демонстрации работы структуры lrstream.
При компиляции и сборке программы исполняемый файл будет носить то же имя, что и данная папка. Рекомендуемое расположение исполняемого файла - папка build/bin.

Собственный поток и манипулятор, позволяющий ограничивать число знаков после запятой для чисел типа LongReal или double (long double и float).

Объект, реализующий "псевдопоток" для вывода в поток типа ostream объектов того же типа
и чисел типа LongReal или double с одним и тем же манипулятором. Манипулятор ограничивает количество
знаков после запятой у чисел типа LongReal или double. Может применяться в методах, где используется
псевдоним decimal, при этом манипулятор будет единственным для обоих типов (LongReal и double).
